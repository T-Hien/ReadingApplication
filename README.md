# 📚 Build Reading App
Ứng dụng đọc sách điện tử (EPUB, PDF), hỗ trợ ghi chú và quản lý thư viện cá nhân.

## 🚀 Công nghệ sử dụng
- Android (Kotlin)
- Spring Boot
- MySQL

## ✨ Tính năng
- Đăng nhập/Đăng ký
- Quản lý thư viện
- Đọc sách, ghi chú, đánh dấu
